# witcher_songs_project
# description
Данный корпус содержит в себе 51 текст современной самодеятельной песни, в которой трансформируется художественный мир и образы цикла книг А. Сапковского "Ведьмак". Тексты песен преобразуются, возвращается список лемм слов для подсчета количества уникальных лемм в тексте, аккумулируются наиболее частые леммы в тексте, создается словарь глаголов. Помимо этого, есть анализ разнообразия словарного запаса каждого тексте(?), вычисляется самое длинное слово, подсчитывается количество слов каждой части речи в тексте, считается упоминание персонажей или заданных слов, связанных с ними. Выводится соотношение цветов и персонажей цикла "Ведьмак" (?)

# project_structure
witcher_songs_project/corpus - содержит 51 текстовый файл с самодеятельными песнями по "Ведьмаку"
witcher_songs_project/corpus/poem01...poem51(?правильно так будет обозначить?) - текстовый файл с текстом песни
witcher_songs_project/data/metadata.csv - файл с информацией о текстах в папке corpus
witcher_songs_project/README.md - файл с описанием проекта
witcher_songs_project/text_utils.py - файл с кодом из пфункций, работающих с текстом в txt-файлах
witcher_songs_project/main.py - файл с кодом, выполняющим все функции и оформляющим их
witcher_songs_project/file_utils.py - файл с кодом, работающим с открытием и созданием папок или csv- и txt-файлов
